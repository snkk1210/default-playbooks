common
=========

## これは何？

基本的な OS 設定用の Playbook です。

- 開発パッケージインストール
- 各種パッケージアップデート
- epel リポジトリインストール
- selinux ライブラリインストール
- selinux 無効化
- firewalld 無効化
- 運用パッケージインストール
- ホストネーム設定
- カーネルパラメータ設定
- limit.conf 設定
- timezone 設定
- history 設定
- /etc/hosts にホストネームを追加
